Dynamically Loading Elements & Modules Into Polymer

Dynamically importing Polymer elements

Let's say we have an element <foo-baz> which contains the below button. In this page, we also have a (currently) unknown <polymer-cat> element we would like to load on-demand.

Click the import button above to dynamically import polymer-cat.html. Here we are using Polymer.import to import our element as follows: 

  Polymer('foo-baz', {
    importElement: function(){
      Polymer.import(['polymer-cat.html'], function() {
            // called when our import is fully loaded
            // including any assets like CSS.
      });
    }
  });

Dynamically loading ES6 Modules / Scripts using ES6 Module Loader

Imagine we have <div id="foo"> and a Polymer element <foo-bar>. <div id="foo"> has no joy, unless we dynamically load Lodash successfully:

the foo has no joy.

When you click the button, <foo-bar> dynamically loads Lodash and injects the updated text into "foo" with help from the _.contains() utility.

How does this work?

Here we're just using the System.import method from our ES6 Module Loader to import the script as follows:

  System.import(base + 'lodash/dist/lodash.min').then(function(){
        var foo = document.getElementById("foo");
        var msg = 'the foo has joy';
        foo.innerHTML = msg + '. Does it _.contains() foo?..' + _.contains(msg, 'foo');
  });

Initializing a Polymer element once dynamic dependencies are loaded

In order to initialize elements only *after* dynamic script dependencies are loaded, we can define a require helper for Polymer (using the polyfilled System.import) as follows:

  Polymer.require = function(tag, deps, func) {
    System.import(deps).then(function() {
      Polymer(tag, func.apply(this, arguments));
    });
  };

We can then use this in our <foo-barbaz> Polymer element to dynamically load Lodash and then return the object defining the methods for our element. If this works, foo2 should be happy:

The code for this is defined as follows: 

  <polymer-element name="foo-barbaz">
    <script> 
      Polymer.require('foo-barbaz', 'lodash.min', function() {
        return {
          ready: function() {
            var foo = document.getElementById("foo2");
            var msg = 'the foo2 has joy';
            foo.innerHTML = msg + '. Does it _.contains() foo?..' + _.contains(msg, 'foo');
          }
        };
      });
    </script>
  </polymer-element>
